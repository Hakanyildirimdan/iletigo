# İletigo - Mutabakat Yönetim Sistemi

Next.js ile geliştirilmiş modern mutabakat yönetim sistemi.

## Özellikler

- 🔐 Güvenli kullanıcı girişi
- 📱 Responsive tasarım
- 🎨 Modern UI/UX
- ⚡ Hızlı performans

## Geliştirme

```bash
# Bağımlılıkları yükle
npm install

# Geliştirme sunucusunu başlat
npm run dev

# Production build
npm run build
npm start
```

## Teknolojiler

- Next.js 14
- TypeScript
- Tailwind CSS
- React 18
